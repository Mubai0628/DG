````markdown
## 1. 本阶段最终目标

完成 v0.11 后，必须具备一条窄口径 App-side approved execution MVP：

```text
Imported / validated patch proposal
→ diff/audit/approval summary
→ explicit App approval receipt
→ Tauri approved apply command
→ user workspace file write under strict path guard
→ summary-only apply event
→ rollback checkpoint / preimage
→ Tauri approved rollback command
→ summary-only rollback event
→ Event Log / Replay can show the chain
```
````

本阶段必须仍然保持：

```text
no auto-apply
no model auto-execution
no Git/shell
no arbitrary filesystem write
no App live DeepSeek execution
no PermissionLease production broad grant
no raw content in events
no API key exposure
```

# DW-P0O-004 — Tauri Approved User Workspace Apply Command

## Goal

Add a narrow Tauri command that can apply approved changes to a user workspace.

This is the first App-side user workspace write path.

It must be guarded, auditable, and summary-only.

## Scope

Add:

- Tauri/Rust approved apply command
- TypeScript wrapper
- App disabled/controlled wiring
- tests
- docs

## Command name

Use a fixed allowlisted command name, for example:

```text
apply_approved_user_workspace_patch
```

If project naming convention suggests another name, use existing convention but keep it explicit and narrow.

## Tauri command input

Must include:

- `workspaceRoot`
- `workspaceRootRef`
- `receipt`
- `operations`
- `proposalSummary`
- `validationSummary`
- `auditSummary`
- `approvalSummary`
- `maxFiles`
- `maxBytes`

Operations:

- `path`
- `changeKind: "create" | "update" | "delete"`
- `content?` for create/update
- `expectedBeforeHash?`
- `expectedExistsBefore?`

## Path safety

Command must block:

- absolute operation path
- Windows drive path
- UNC path
- parent traversal
- null/newline
- shell metacharacters
- URL/query-like paths
- `.git`
- `.env`
- `node_modules`
- `dist`
- `target`
- `.tmp`
- generated artifacts
- secret-like filenames
- symlink/junction/reparse escape
- directory delete
- recursive delete

Workspace root must block:

- drive root
- home/profile root
- system root
- system temp root itself
- nonexistent root unless specifically created by test fixture
- path that cannot canonicalize

## Content safety

Block:

- fake API key
- Bearer token
- Authorization
- private key markers
- rawPrompt/rawDom/rawCsv/rawScreenshot/clipboard markers
- binary-looking content unless explicitly allowed later
- content over maxBytes
- operation count over maxFiles

## Checkpoint / backup

Before applying update/delete:

- capture preimage in local checkpoint storage.
- checkpoint may contain raw preimage content.
- checkpoint must be stored only under workspace private workbench area, e.g.:

```text
<workspace>/.deepseek-workbench/checkpoints/<checkpointId>.json
```

- checkpoint content must never be shown in UI summary.
- checkpoint content must never be written to EventStore.
- checkpoint must include:
  - path
  - existedBefore
  - preimageHash
  - preimageBytes
  - changeKind
  - content stored for rollback
- ensure `.deepseek-workbench/.gitignore` or equivalent protects local checkpoint artifacts if appropriate.

## Apply result

Return summary-only:

- `applyId`
- `checkpointId`
- `workspaceRootRef`
- operation counts
- filesCreated/Updated/Deleted
- bytesWritten
- hashes
- warning codes
- `eventPreview`
- no raw content
- no preimage

## Event

Either:

- write summary apply event in command, or
- return eventPreview and let P0O-006 wire summary event write.

For P0O-004, at minimum return summary-only eventPreview with `notWritten: true`.

## App wrapper

Add/update:

- `app/src/desktop-flow.ts`
- App wrapper function for command, fixed allowlist only.

No generic command runner.

## App UI

At this subtask, do not enable final apply yet unless all gating is complete.

Add controlled state / disabled placeholder if needed.

## Tests

Rust/Tauri tests:

- safe create in temp workspace
- safe update with checkpoint
- safe delete with checkpoint
- unsafe path blocked
- .env/.git blocked
- symlink/reparse if testable
- wrong receipt blocked
- wrong typed confirmation blocked

- summary result contains no raw content

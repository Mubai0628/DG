## Current Capabilities

- Convert: `web_table_to_csv` remains the real conversion flow.
- Event Log / Replay: summary-only event replay remains available.
- Record Draft Event: App/Tauri local summary-event write path remains
  available.
- Approved execution: App-side approved apply and rollback exist behind human
  approval receipt and typed confirmation.
- Git/shell safe lanes: fixed read-only Git lanes and fixed shell verification
  templates remain summary-only.
- Live proposal generation: App live DeepSeek proposal generation remains
  explicit opt-in and cannot auto-apply.
- E2E task flow: live proposal, approved apply, verification, rollback, and
  replay surfaces remain connected without expanding execution authority.
- MVP hardening/recovery: recovery, replay, regression, and smoke hardening are
  in place.
- Production memory/project knowledge: workspace-local, human-reviewed
  `policy`, `project_fact`, and `pitfall` summaries can be committed, recalled,
  revoked, expired, replayed, and redaction-audited.

任务目标：完成所有小阶段任务。

本阶段目标是完成并发布：

`v0.18.0-mcp-readonly-connection-mvp-rc.1`

主题：

**MCP Read-only Connection MVP, no tool invocation**

v0.17 已完成 Capability Host descriptor preview。v0.18 在此基础上只打开一条极窄的 **MCP read-only discovery / connection** 路径，用来发现 MCP server 的 resources / prompts / tools metadata，并纳入 Capability Broker 与 App 只读 surface。

本阶段仍然禁止：

- MCP tool invocation
- MCP mutating tools
- MCP resource content read by default
- plugin code execution
- skill runtime execution
- broad PermissionLease
- arbitrary process spawn
- arbitrary shell command
- App-side hidden execution
- native bridge
- desktop action
- autonomous agent tool execution

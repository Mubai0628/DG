# v0.19 — MCP Tool Invocation Proposal / Approval Design 阶段提示词

任务目标：完成所有小阶段任务。

本阶段目标是完成并发布：

`v0.19.0-mcp-tool-invocation-proposal-rc.1`

主题：

**MCP Tool Invocation Proposal / Approval Design, no real tool invocation**

v0.18 已完成 MCP read-only connection / discovery。v0.19 在此基础上只建立 **MCP tool invocation proposal / approval / simulation** 体系，让 MCP tools 进入 Capability Broker、risk policy、approval draft、dry/simulated result 与 App read-only proposal surface，但仍然不执行真实 `tools/call`。

本阶段仍然禁止：

- real MCP `tools/call`
- mutating MCP tools
- MCP resource content read by default
- plugin code execution
- skill runtime execution
- arbitrary process spawn
- arbitrary shell command
- App-side hidden MCP invocation
- App-side automatic tool invocation
- broad PermissionLease
- native bridge
- desktop action
- autonomous agent tool execution
- raw tool args / raw output / raw prompt / raw response / API key in events or App summaries

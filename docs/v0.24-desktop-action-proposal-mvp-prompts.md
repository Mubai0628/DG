# v0.24 — Desktop Action Proposal MVP 阶段提示词

任务目标：完成所有小阶段任务。

本阶段目标是完成并发布：

`v0.24.0-desktop-action-proposal-mvp-rc.1`

主题：

**Desktop Action Proposal MVP, no execution**

v0.23 已经完成 Desktop Observer MVP：App 可以 user-triggered 观察窗口 / 应用 / 显示器等 metadata，并将 observer evidence refs 接入 context / agent dossiers。v0.24 的目标不是执行桌面动作，而是把未来桌面动作建模成 **proposal / risk / approval draft / dry-run simulation / audit / App read-only surface**。

本阶段绝对禁止真实桌面动作执行。

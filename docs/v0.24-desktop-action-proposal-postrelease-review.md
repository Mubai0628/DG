# v0.24 Desktop Action Proposal Post-release Review

Release tag: `v0.24.0-desktop-action-proposal-mvp-rc.1`

Release title: `Desktop Action Proposal MVP, no desktop action`

Release status: prerelease published after local full gates, main CI, and tag CI passed.

## Current Capabilities

- Desktop Observer metadata evidence: user-triggered, summary-only foreground/window/app/display evidence refs.
- Desktop Action Proposal schema: structured proposal contract for future desktop actions.
- target/window/app/screen metadata validation: deterministic validation of observed target references and stale/unsafe metadata.
- action risk classifier: fixed risk classification that blocks unsafe or unsupported action classes.
- dry-run/simulated result: summary-only desktop action simulation with no real desktop action.
- broker planning integration: Capability Broker planning refs for desktop action proposals without execution.
- App read-only action proposal surface: proposal paste/preview only with disabled execution placeholders.
- privacy/redaction audit: blocks raw screenshot, raw OCR, secret markers, and unsafe raw fields from proposal summaries.

## Disabled Capabilities

- Real desktop action.
- Click/type/select execution.
- Drag/drop execution.
- Clipboard write.
- File dialog automation.
- Screen recording.
- Hidden background capture.
- Native bridge broad action.
- Remote control.
- Dynamic agent desktop control.
- Autonomous desktop agent.

## Release Hygiene

- No raw screenshot persistence by default.
- No raw OCR persistence.
- No raw prompt/source/diff/API key/event payload leakage.
- Desktop action proposals remain summary-only and non-executing.
- App Shell still cannot perform desktop actions, invoke a broad native bridge, or issue broad PermissionLeases.

## Next Stage

P1C opens an extremely narrow, human-approved desktop action execution path for observed window focus/raise/activate actions only. It does not enable click/type/select, clipboard write, file dialog automation, remote control, hidden background action, broad native bridge, or autonomous desktop agents.

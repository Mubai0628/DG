任务目标：完成所有小阶段任务。

`v0.26.0-desktop-action-expansion-proposal-rc.1`

**Desktop Action Expansion Proposal Gate, no click/type execution**

## 1. 本阶段最终目标

完成 v0.26 后，必须具备一条 **desktop action expansion proposal gate**：

```text
Desktop observer evidence
→ expanded desktop action proposal
→ target freshness / screen mismatch validation
→ sensitive target / destructive UI risk classifier
→ action sequence simulation
→ approval draft / broker planning summary
→ App read-only proposal surface
→ privacy / redaction audit
→ no click/type/clipboard/file-dialog execution
```

本阶段只建模未来扩展动作，不执行这些动作。

明确支持 proposal / simulation 的动作类别：

- `click_target`
- `type_text`
- `select_option`
- `keyboard_shortcut`
- `clipboard_write`
- `file_dialog_select`
- `drag_drop`
- `scroll`
- `wait_for_state`
- `focus_observed_window` / `raise_observed_window` / `activate_observed_window` 可以作为已有 v0.25 窄口径执行能力的 reference，但本阶段不扩大执行。

本阶段必须仍然保持：

```text
no real click
no real type
no real select
no clipboard write
no file dialog automation
no drag/drop execution
no hidden capture
no screen recording
no remote control
no broad native bridge
no dynamic agent desktop control
no App-side arbitrary desktop action
```

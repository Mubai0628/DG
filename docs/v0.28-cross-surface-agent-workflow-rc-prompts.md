# v0.28 Cross-surface Agent Workflow RC 阶段提示词

任务目标：完成所有小阶段任务。

阶段目标：完成 `v0.28.0-cross-surface-agent-workflow-rc.1`，把已发布的 coding workflow、project knowledge、MCP read-only tool、plugin/skill metadata、fixed multi-agent、desktop observer、desktop action proposal、approved workspace apply/rollback、Git/shell verification safe lanes、approved desktop action lane 串成一个可演示的 cross-surface agent workflow。

本阶段不是继续扩大执行半径，而是做整合、演示、replay、audit、redaction、manual QA。仍然禁止 broad automation、dynamic agent bidding、arbitrary tool use、arbitrary Git/shell、broad native bridge、clipboard/file dialog/drag-drop execution、background/hidden desktop action、autonomous desktop agent。

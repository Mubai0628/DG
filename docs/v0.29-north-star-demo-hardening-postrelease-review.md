## Release Hygiene

- Events and replay remain summary-only.
- Raw source, raw diff, raw CSV, raw prompt, raw response, reasoning content,
  and API keys are not allowed in events, replay, telemetry, or App surfaces.
- Conformance and eval artifacts remain generated artifacts and are not
  committed.

execution scope. The next phase should focus on MCP read-only consistency,
plugin/skill sandbox escape checks, Capability Broker policy enforcement,
PermissionLease / approval receipt consistency, replay completeness, redaction
audit, an App read-only audit surface, and golden safety smoke.

## What Works

- `web_table_to_csv` Convert remains the active desktop workflow.
- Event Log / Replay shows summary event and draft information.
- Chat / Run Canvas is draft-only and local-state-only.
- Local Run Draft Preview produces preview-only local summaries.
- Control Plane Projection derives read-only state from event summaries.
- Context Cart / Rules Ledger visualization shows metadata-only context
  boundaries.
- Agent Route Preview shows fixed route summaries only.
- Capability Plan Preview shows planning-only descriptor summaries.
- Patch Proposal UI Bridge shows patch and audit summaries without apply.
- Approval / Diff / Audit Surfaces are read-only.
- Memory Inspector is read-only and not connected to persistence.
- Memory Recall Preview shows volatile-tail recall refs only.
- Bridge Proposal Preview is disabled when no live bridge proposal exists.

## What Is Intentionally Disabled

- Real DeepSeek chat.
- Real ControlPlaneRun creation from the UI.
- Patch apply.
- Git execution.
- Shell execution.
- Capability invocation.
- PermissionLease issuing.
- Memory commit, revoke, or expire UI.
- MCP, plugin, or skills runtime.
- Native bridge, `nativeMessaging`, localhost bridge, or extension auto-send.
- Desktop action.

## Boundary Lock

The App Shell can display draft-only, read-only, preview-only, or planning-only
surfaces. It cannot execute those plans. Future P0H work must preserve:

1. controlled creation before execution,
2. summary-only state and events,
3. explicit approval before any future side effect,
4. safe source boundaries for workspace summaries,
5. no raw prompt, raw source code, raw DOM, raw CSV, API key, authorization
   header, environment value, screenshot, clipboard content, or full memory
   content in event payloads or preview surfaces.

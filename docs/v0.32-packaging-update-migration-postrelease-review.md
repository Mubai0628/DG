- No destructive migration.
- No silent data deletion.
- No cloud sync.
- No telemetry upload.
- No broad native bridge.
- No arbitrary desktop automation.
- No mutating MCP tools.
- No arbitrary plugin / skill execution.
- No autonomous arbitrary tool execution.
- No arbitrary Git / shell execution.
- No broad PermissionLease.
- No hidden background desktop control or remote control.

## Release Hygiene

- Events and docs must not persist raw prompt, raw source, raw diff, raw model
  response, reasoning content, raw CSV, raw DOM, screenshots, API keys, or
  Authorization values.
- Generated artifacts remain ignored: `app/dist`, `app/src-tauri/target`,
  `runtime/dist`, `browser-extension/dist`, `conformance/results`, `.tmp`,
  logs, coverage, screenshots, and release zips.
- Conformance and eval artifacts are local diagnostics and are not release
  payloads by default.

## P1K Handoff

P1K should not expand execution authority. It should convert the current North
Star Beta baseline into v1.0 candidate readiness evidence: security audit
matrix, capability boundary matrix, golden regression dashboard, final artifact
hygiene, data migration dry-run review, onboarding polish, rollback guidance,
and final North Star manual QA.

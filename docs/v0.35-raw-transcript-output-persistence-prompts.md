# v0.35 Raw Transcript / Output Persistence 阶段提示词

任务目标：完成所有小阶段任务。

本阶段目标是完成 **Raw Transcript / Output Persistence** 基础设施，为后续 v0.36 任意 Shell / Command Broker、v0.40 Autonomous Loop、v0.41 Full Access Session 提供可审计、可保留、可删除、可导出、默认脱敏的 transcript 层。

本阶段不是任意命令执行阶段。不要打开 arbitrary shell，不要自动执行命令，不要 auto apply，不要递归删除，不要 Git commit/push，不要 autonomous loop，不要 full access execution。

- transcript 默认 redacted summary-only。
- raw output 只能 explicit opt-in。
- raw transcript 不等于 full access。
- raw transcript 不允许自动启用 shell。
- raw transcript 不允许自动启用 command execution。

- transcript events 仍 summary-only。

- API key leakage

- reasoning_content persistence
- source/diff leakage
- terminal escape sequences
- ANSI/control chars
- binary output
- huge output / disk bloat
- command injection in transcript rendering
- HTML/script injection
- path leakage
- workspace privacy
- stale retention
- export leakage
- delete bypass
- replay accidental raw display
- raw transcript in release artifacts

## Implementation gate 必须可测试

分类：

- schema safety
- redaction safety
- raw opt-in safety
- storage safety
- retention/delete/export safety
- App viewer safety

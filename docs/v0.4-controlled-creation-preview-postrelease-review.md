## What Works

- `web_table_to_csv` Convert remains the real conversion flow.
- Event Log / Replay shows summary event and draft information.
- Workspace Index summary bridge accepts safe summary JSON only.
- Chat / Run Canvas is draft-only.
- Run Draft Preview produces local preview summaries.
- Record Draft Event can append one summary-only local event after explicit user
  action.
- Context Assembly Preview shows local context summaries without prompt
  assembly.
- Runtime Agent Route Preview helper provides fixed route summaries.
- Runtime Capability Plan Preview helper provides planning-only descriptor
  summaries.
- Patch Proposal Creation Preview creates local patch proposal summaries.
- Patch Proposal / Diff UI Bridge displays proposal and audit summaries without
  apply.
- Approval / Diff / Audit Surfaces are read-only.
- Memory Inspector remains read-only and not connected to persistence.
- Runtime Memory Recall Preview helper returns volatile-tail recall refs only.
- Bridge Proposal Preview remains disabled when no live bridge proposal exists.

## What Is Intentionally Disabled

- Real DeepSeek chat.
- Real run execution.
- Patch apply.
- Git execution.
- Shell execution.
- Capability invocation.
- PermissionLease issuing.
- Memory commit, revoke, or expire UI.
- MCP, plugin, or skills runtime.
- Native bridge, `nativeMessaging`, localhost bridge, or extension auto-send.
- Desktop action.

## Boundary Lock

The only allowed v0.4 side effect is explicit user-triggered Record Draft Event,
which appends a summary-only local draft event. All other coding workflow
surfaces remain read-only, preview-only, local-only, or planning-only.

Future P0I work must preserve:

1. validation before approval,
2. approval drafts before approval execution,
3. virtual apply before any real apply,
4. summary-only state and events,
5. no raw prompt, raw source code, raw DOM, raw CSV, API key, authorization
   header, environment value, screenshot, clipboard content, or full memory
   content in event payloads or preview surfaces.
